# Evidence

This folder contains sanitised or non-sensitive evidence supporting the Data Analytics Platform Management project.

## Evidence Included

| Evidence | Purpose |
|---|---|
| `evidence-summary.md` | Explains how the project is evidenced safely without exposing confidential operational information |
| `qlik-deploy-administer-qlik-sense.pdf` | Supports Qlik platform administration and management experience |

## Evidence Approach

This project does not include internal SOPs, real platform screenshots, user data, client information, ticket references, internal system URLs, licence counts, or confidential operational details.

The main project evidence is represented through:

- High-level workflow diagrams
- Access request templates
- JML checklist templates
- Licence tracking templates
- Sanitised evidence summaries
- Relevant platform training evidence

## Confidentiality Note

All evidence is included only where it is safe for public portfolio use. Internal operational documents and confidential platform details are excluded.
